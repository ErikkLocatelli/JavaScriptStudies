Web Storage Api 

Armazenamento de dados do lado do cliente. 

O client side storage pode ser utilizado para diferentes finalidades, tais como: 

    	1. Personalizar as preferências do site (tema claro/escuro pré definido em uma visita anterior)
        2. Atividade anterior feita no site (Ex: Salvar o carrinho de uma loja montado em uma visita)
        3. Salvar, localmente documentos gerados durante o uso de alguma aplicação 

Tipos de client side storage: 

    1. Cookies 
    2. Local storage 
    3. Session Storage 

1. Local Storage: ele guarda os dados de cada aplicação de forma isolada, baseando-se na URL base da página. Todos esses dados continuam armazenados mesmo se o navegador for fechado e aberto novamente. 

2. Session Storage: o armazenamento funciona da mesma forma nele, mas os dados ficam disponíveis apenas durante a sessão na página. Portanto, quanto a página do navegador estiver aberta, as informações e dados continuaram nele.
